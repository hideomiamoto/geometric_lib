# Math formulas
The main purpose of the project is to learn how to use git and github. The formulas below are implemented as programs and are used for tests.

### Area
- Circle: S = πR²
- Rectangle: S = ab
- Triangle: S = ½ab
- Square: S = a²

### Perimeter
- Circle: P = 2πR
- Rectangle: P = 2a + 2b
- Triangle: P = a + b + c
- Square: P = 4a
